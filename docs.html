GrubY

Modern Ruby Telegram toolkit with Bot API + TDLib (tdjson) support.

Core Engines

  • GrubY::Client for Bot API
  • GrubY::TDLib::Client for TDLib runtime
  • GrubY::TDLib::ClientManager for multi-client flow

TDLib Enhancements

  • Dedicated TdJson wrapper
  • Decorator handlers: initializer, finalizer, on_message
  • Queue workers and timeout support

Quick Start

client = GrubY::TDLib::Client.new(
  api_id: ENV.fetch("TD_API_ID").to_i,
  api_hash: ENV.fetch("TD_API_HASH"),
  tdjson_path: ENV["TDJSON_PATH"],
  bot_token: ENV["TD_BOT_TOKEN"]
)

client.on("clientReady") { puts "ready" }
client.on_message { |message| puts message.inspect }
client.run
